Michi REST permite la búsqueda, creación, actualización y eliminación de razas de gatos experimentals por la Fédération Internationale Féline (FIF) además de razas experimentales.

Endpoints

User

GET

Traer todos los usuarios

http://127.0.0.1:3000/api/users

Traer un usuario por su nombre

http://127.0.0.1:3000/api/users/name/:name

Traer un usuario por su ID

http://127.0.0.1:3000/api/users/:id

POST

Crear un usuario

http://127.0.0.1:3000/api/users/

Iniciar Sesión

http://127.0.0.1:3000/api/users/login

PUT

Actualizar un usuario

http://127.0.0.1:3000/api/users/:id

DELETE

Eliminar un usuario

http://127.0.0.1:3000/api/users/:id
